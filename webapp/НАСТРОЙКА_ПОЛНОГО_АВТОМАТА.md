# 🤖 Полностью автоматический деплой

## 🎯 Что это?

**Вы вообще ничего не делаете!** Система работает в фоне и автоматически деплоит любые изменения в Telegram.

---

## ⚡ Быстрая настройка (один раз)

### ШАГ 1: Настройка GitHub

1. Запустите `БЫСТРАЯ_НАСТРОЙКА_GITHUB.bat`
   - Настроит Git и GitHub

2. Подключите Netlify к GitHub:
   - https://app.netlify.com/
   - Add new site → Import from GitHub
   - Выберите репозиторий
   - Base directory: `webapp`
   - Deploy site

### ШАГ 2: Запуск автоматического деплоя

**Вариант А: Запуск вручную (каждый раз при включении компьютера)**

1. Запустите `ЗАПУСТИТЬ_АВТОДЕПЛОЙ_В_ФОНЕ.bat`
2. Всё! Система работает в фоне

**Вариант Б: Автозапуск при старте Windows (РЕКОМЕНДУЕТСЯ)**

1. Нажмите `Win + R`
2. Введите: `shell:startup`
3. Нажмите Enter
4. Скопируйте ярлык файла `АВТОЗАПУСК_ПРИ_СТАРТЕ.bat` в эту папку
5. Готово! Теперь деплой запускается автоматически при включении компьютера

## 📋 Как это работает?

1. **Скрипт работает в фоне** постоянно
2. **Отслеживает изменения** в файлах каждые 3 секунды
3. **При обнаружении изменений**:
   - Автоматически делает `git add`
   - Автоматически делает `git commit`
   - Автоматически делает `git push`
4. **Netlify автоматически** получает изменения и деплоит
5. **Через 1-2 минуты** изменения в Telegram!

## ⚙️ Настройки

Откройте `auto_deploy_background.py` и измените:

```python
deploy_cooldown = 20  # Минимум 20 секунд между деплоями
WATCH_FILES = ['index.html', 'style.css', 'app.js']  # Какие файлы отслеживать
```

## 🛠️ Управление

### Остановить автоматический деплой:

1. Откройте Диспетчер задач (Ctrl+Shift+Esc)
2. Найдите процесс `pythonw.exe` или `python.exe`
3. Завершите процесс

### Перезапустить:

Просто запустите `ЗАПУСТИТЬ_АВТОДЕПЛОЙ_В_ФОНЕ.bat` снова

---

## ❓ Проблемы?

### Деплой не работает

- Проверьте `auto_deploy.log` - там будут ошибки
- Убедитесь, что Git настроен
- Убедитесь, что Netlify подключен к GitHub

### Изменения не появляются

- Подождите 2-3 минуты
- Проверьте логи Netlify на сайте
- Обновите мини-приложение в Telegram

### Скрипт не запускается

- Установите Python: https://www.python.org/
- Установите Git: https://git-scm.com/download/win
