# 🔧 Инструкция по восстановлению style.css

## ⚠️ Проблема
На развернутой версии (Netlify) есть код, который скрывает все элементы приложения.

## ✅ Решение

### Вариант 1: Восстановить из резервной копии (если есть)
Если у вас есть резервная копия или git репозиторий:
```bash
git checkout webapp/style.css
```

### Вариант 2: Скачать с Netlify и исправить
1. Откройте: https://frolicking-youtiao-036d5e.netlify.app/style.css
2. Скопируйте весь код
3. Удалите из него проблемные блоки:
   - Удалите блок с `body > *:not(script) { display: none !important; }`
   - Удалите блок с `[class*="marketplace"]` и т.д.
4. Вставьте исправленный код в `webapp/style.css`
5. Добавьте в конец файла стили MineBoom из `style_mineboom_only.css`

### Вариант 3: Использовать текущий файл
Текущий файл `style.css` содержит только стили MineBoom и НЕ содержит проблемного кода.
Но в нем отсутствуют остальные стили приложения.

## 📝 Что нужно сделать после восстановления:
1. Убедитесь, что в файле НЕТ кода:
   ```css
   body > *:not(script) {
       display: none !important;
   }
   ```
2. Добавьте стили MineBoom в конец файла (если их там нет)
3. Задеплойте исправленный файл на Netlify

## 🚀 После исправления:
После того как восстановите правильный `style.css`, задеплойте его на Netlify, и приложение должно заработать.
