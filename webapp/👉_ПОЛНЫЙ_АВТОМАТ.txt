📋 НАСТРОЙКА (ОДИН РАЗ):

┌─────────────────────────────────────────────────────────────┐
│ ШАГ 1: Настройка GitHub                                     │
└─────────────────────────────────────────────────────────────┘

1. Запустите: БЫСТРАЯ_НАСТРОЙКА_GITHUB.bat
   ✓ Настроит Git
   ✓ Подключит к GitHub

2. Подключите Netlify к GitHub:
   → https://app.netlify.com/
   → Add new site → Import from GitHub
   → Выберите репозиторий
   → Base directory: webapp
   → Deploy site

┌─────────────────────────────────────────────────────────────┐
│ ШАГ 2: Запуск автоматического деплоя                        │
└─────────────────────────────────────────────────────────────┘

ВАРИАНТ А: Запуск вручную
→ Запустите: ЗАПУСТИТЬ_АВТОДЕПЛОЙ_В_ФОНЕ.bat

ВАРИАНТ Б: Автозапуск (РЕКОМЕНДУЕТСЯ)
1. Win+R → shell:startup
2. Скопируйте ярлык: АВТОЗАПУСК_ПРИ_СТАРТЕ.bat
3. Готово! Теперь запускается автоматически!

   → git push (автоматически)

5. Деплоит на сайт (автоматически)
6. Через 1-2 минуты - в Telegram!
