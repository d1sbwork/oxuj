# 🚀 Автоматический деплой в Telegram

## Вариант 1: Автоматический деплой через GitHub (РЕКОМЕНДУЕТСЯ)

### Настройка один раз:

1. **Создайте репозиторий на GitHub** (если еще нет):
   - Зайдите на https://github.com/new
   - Создайте новый репозиторий (например: `telegram-miniapp`)

2. **Подключите Netlify к GitHub**:
   - Зайдите на https://app.netlify.com/
   - Нажмите "Add new site" → "Import an existing project"
   - Выберите "GitHub"
   - Выберите ваш репозиторий
   - Настройки:
     - **Base directory**: `webapp` (или оставьте пустым, если файлы в корне)
     - **Build command**: оставьте пустым
     - **Publish directory**: `.` (или `webapp`, если указали Base directory)
   - Нажмите "Deploy site"

3. **Готово!** Теперь при каждом коммите в GitHub, Netlify автоматически обновит сайт.

### Как использовать:

1. **Отредактируйте файлы** в папке `webapp`:
   - `index.html`
   - `style.css`
   - `app.js`
   - и т.д.

2. **Загрузите на GitHub**:
   - Через GitHub Desktop: Commit → Push
   - Или через веб-интерфейс GitHub

3. **Ждите 1-2 минуты** - Netlify автоматически обновит сайт

4. **Обновите мини-приложение в Telegram** - изменения появятся!

## Вариант 2: Быстрый деплой через Netlify CLI

### Установка (один раз):

1. **Установите Node.js**: https://nodejs.org/
2. **Установите Netlify CLI**:
   ```bash
   npm install -g netlify-cli
   ```
3. **Войдите в Netlify**:
   ```bash
   netlify login
   ```
4. **Подключите к сайту**:
   ```bash
   cd webapp
   netlify link
   ```
   Выберите ваш сайт из списка

### Использование:

1. **Отредактируйте файлы** в папке `webapp`

2. **Запустите скрипт**:
   - Двойной клик на `БЫСТРЫЙ_ДЕПЛОЙ.bat`
   - Или в командной строке:
     ```bash
     cd webapp
     netlify deploy --prod --dir .
     ```

3. **Готово!** Изменения появятся в Telegram через 1-2 минуты

## Вариант 3: Ручной деплой через Netlify веб-интерфейс

1. **Отредактируйте файлы** в папке `webapp`

2. **Загрузите на Netlify**:
   - Зайдите на https://app.netlify.com/
   - Выберите ваш сайт
   - Перейдите в "Deploys"
   - Нажмите "Trigger deploy" → "Deploy site"
   - Или перетащите папку `webapp` в область "Deploy manually"

3. **Готово!** Изменения появятся в Telegram

## ❓ Проблемы?

- **Сайт не обновляется?** Подождите 2-3 минуты, затем обновите мини-приложение в Telegram
- **Ошибка деплоя?** Проверьте, что все файлы сохранены
- **Netlify CLI не работает?** Используйте GitHub Desktop (Вариант 1)
