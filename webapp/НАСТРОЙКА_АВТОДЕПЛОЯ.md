# 🤖 Автоматический деплой при изменениях

## 🎯 Что это?

Система автоматически отслеживает изменения в файлах (`index.html`, `style.css`, `app.js`) и **автоматически деплоит** их в Telegram мини-приложение!

**Просто редактируйте файлы - всё остальное происходит автоматически!**

---

## ⚡ Быстрый старт

### Вариант 1: Через GitHub (РЕКОМЕНДУЕТСЯ) ⭐

#### Настройка (один раз):

1. **Создайте репозиторий на GitHub**:
   - Зайдите на https://github.com/new
   - Создайте репозиторий (например: `telegram-miniapp`)

2. **Инициализируйте Git в проекте**:
   ```bash
   cd C:\Users\Alex\Desktop\Case
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/ВАШ_ЛОГИН/telegram-miniapp.git
   git push -u origin main
   ```

3. **Настройте Git** (если еще не настроен):
   ```bash
   git config --global user.name "Ваше Имя"
   git config --global user.email "ваш@email.com"
   ```

4. **Подключите Netlify к GitHub**:
   - Зайдите на https://app.netlify.com/
   - Add new site → Import an existing project
   - Выберите GitHub → ваш репозиторий
   - Настройки:
     - **Base directory**: `webapp`
     - **Build command**: (оставьте пустым)
     - **Publish directory**: `.`
   - Deploy site

5. **Готово!** Теперь всё автоматически!

#### Использование:

1. **Запустите мониторинг**:
   - Двойной клик на `АВТОМАТИЧЕСКИЙ_ДЕПЛОЙ_ПРИ_ИЗМЕНЕНИЯХ.bat`
   - Выберите вариант 1 (GitHub)

2. **Редактируйте файлы**:
   - Откройте `index.html`, `style.css`, `app.js` в редакторе
   - Вносите изменения
   - **Сохраните файл** (Ctrl+S)

3. **Всё!** Скрипт автоматически:
   - Обнаружит изменения
   - Сделает git commit
   - Отправит в GitHub
   - Netlify автоматически обновит сайт
   - Через 1-2 минуты изменения появятся в Telegram!

## 📋 Как это работает?

1. **Скрипт `auto_deploy.py`** отслеживает изменения в файлах:
   - `index.html`
   - `style.css`
   - `app.js`

2. **При обнаружении изменений**:
   - Вычисляет хеш файла
   - Сравнивает с предыдущим значением
   - Если изменился → запускает деплой

3. **Деплой**:
   - **GitHub режим**: `git add` → `git commit` → `git push`
   - **Netlify режим**: `netlify deploy --prod`

4. **Результат**:
   - Изменения появляются в Telegram через 1-2 минуты!

## ⚙️ Настройки

Откройте `auto_deploy.py` и измените:

```python
WATCH_FILES = ['index.html', 'style.css', 'app.js']  # Какие файлы отслеживать
GITHUB_MODE = False  # True = GitHub, False = Netlify CLI
```

## 🛠️ Решение проблем

### Скрипт не запускается

- **Python не найден**: Установите Python с https://www.python.org/
- **Netlify CLI не найден**: `npm install -g netlify-cli`
- **Git не найден**: Установите Git с https://git-scm.com/

### Деплой не работает

- **GitHub режим**: Проверьте, что репозиторий подключен (`git remote -v`)
- **Netlify режим**: Проверьте, что сайт подключен (`netlify status`)
- **Ошибка авторизации**: Выполните `netlify login` или настройте Git

### Изменения не появляются в Telegram

- Подождите 2-3 минуты (Netlify нужно время на обновление)
- Обновите мини-приложение в Telegram (закройте и откройте снова)
- Проверьте логи Netlify на сайте
