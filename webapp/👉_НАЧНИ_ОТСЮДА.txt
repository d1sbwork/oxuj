📋 ЧТО НУЖНО СДЕЛАТЬ:

┌─────────────────────────────────────────────────────────────┐
│ ШАГ 1: Проверка (2 минуты)                                 │
└─────────────────────────────────────────────────────────────┘

1. Запустите: ПРОВЕРКА_НАСТРОЙКИ.bat
   → Проверит, что всё установлено
   → Покажет, что нужно исправить

2. Если есть ошибки - исправьте их:
   • Python не установлен? → https://www.python.org/downloads/
   • Git не установлен? → https://git-scm.com/download/win

┌─────────────────────────────────────────────────────────────┐
│ ШАГ 2: Настройка GitHub (5 минут)                          │
└─────────────────────────────────────────────────────────────┘

1. Запустите: БЫСТРАЯ_НАСТРОЙКА_GITHUB.bat
   → Настроит Git
   → Подключит к GitHub
   → Следуйте инструкциям на экране

2. Если нужно создать репозиторий:
   → Зайдите на https://github.com/new
   → Создайте репозиторий
   → Скопируйте URL
   → Вставьте в скрипт

┌─────────────────────────────────────────────────────────────┐
│ ШАГ 3: Подключение Netlify (5 минут)                       │
└─────────────────────────────────────────────────────────────┘

1. Откройте: https://app.netlify.com/
2. Add new site → Import from GitHub
3. Выберите ваш репозиторий
4. Настройки:
   Base directory:    webapp
   Publish directory: .
5. Deploy site

┌─────────────────────────────────────────────────────────────┐
│ ШАГ 4: Запуск автоматического деплоя (1 минута)            │
└─────────────────────────────────────────────────────────────┘

1. Запустите: ЗАПУСТИТЬ_АВТОДЕПЛОЙ_В_ФОНЕ.bat
   → Запустит систему в фоне

2. Или настройте автозапуск:
   → Win+R → shell:startup
   → Скопируйте ярлык АВТОЗАПУСК_ПРИ_СТАРТЕ.bat

┌─────────────────────────────────────────────────────────────┐
│ ШАГ 5: Проверка (2 минуты)                                 │
└─────────────────────────────────────────────────────────────┘

1. Запустите: ПРОВЕРКА_НАСТРОЙКИ.bat снова
   → Должно показать: ✅ ВСЁ ОТЛИЧНО!

2. Протестируйте:
   → Откройте index.html
   → Измените что-то
   → Сохраните (Ctrl+S)
   → Подождите 1-2 минуты
   → Проверьте Telegram
