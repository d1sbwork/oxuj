📋 ЧТО ДЕЛАТЬ:

┌─────────────────────────────────────────────────────────────┐
│ ШАГ 1: Настройка (один раз)                                 │
└─────────────────────────────────────────────────────────────┘

1. Запустите: БЫСТРАЯ_НАСТРОЙКА_GITHUB.bat
   - Это настроит Git и подключит к GitHub
   - Следуйте инструкциям на экране

2. Подключите Netlify к GitHub:
   - Зайдите на https://app.netlify.com/
   - Add new site → Import from GitHub
   - Выберите ваш репозиторий
   - Base directory: webapp
   - Publish directory: .
   - Deploy site

┌─────────────────────────────────────────────────────────────┐
│ ШАГ 2: Запуск автоматического деплоя                        │
└─────────────────────────────────────────────────────────────┘

1. Запустите: АВТОМАТИЧЕСКИЙ_ДЕПЛОЙ_ПРИ_ИЗМЕНЕНИЯХ.bat
2. Выберите вариант 1 (GitHub)
3. Оставьте окно открытым

┌─────────────────────────────────────────────────────────────┐
│ ШАГ 3: Работа (каждый раз)                                  │
└─────────────────────────────────────────────────────────────┘

1. Откройте файлы в редакторе:
   - index.html
   - style.css
   - app.js

2. Вносите изменения

3. Сохраните файл (Ctrl+S)

4. ВСЁ! Скрипт автоматически:
   ✓ Обнаружит изменения
   ✓ Отправит в GitHub
   ✓ Netlify обновит сайт
   ✓ Через 1-2 минуты изменения в Telegram!

❓ ПРОБЛЕМЫ?

• Скрипт не запускается?
  → Установите Python: https://www.python.org/

• Git не найден?
  → Установите Git: https://git-scm.com/download/win

• Деплой не работает?
  → Проверьте, что Netlify подключен к GitHub
  → Запустите БЫСТРАЯ_НАСТРОЙКА_GITHUB.bat снова

• Изменения не появляются?
  → Подождите 2-3 минуты
  → Закройте и откройте мини-приложение в Telegram
