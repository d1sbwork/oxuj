📌 ШАГ 0: ПРОВЕРКА (2 минуты)

1. Запустите: ПРОВЕРКА_НАСТРОЙКИ.bat
   → Проверит всё необходимое
   → Покажет, что нужно установить

2. Если есть ошибки:
   • Python не установлен?
     → https://www.python.org/downloads/
     → При установке отметьте "Add Python to PATH"
   
   • Git не установлен?
     → https://git-scm.com/download/win
     → Установите с настройками по умолчанию

3. Запустите проверку снова, пока не будет: ✅ ВСЁ ОТЛИЧНО!

═══════════════════════════════════════════════════════════════

📌 ШАГ 1: НАСТРОЙКА GITHUB (5 минут)

1. Запустите: БЫСТРАЯ_НАСТРОЙКА_GITHUB.bat
   → Двойной клик на файле

2. Следуйте инструкциям:
   ✓ Введите ваше имя (для Git)
   ✓ Введите ваш email (для Git)
   
   Если репозиторий не создан:
   → Зайдите на https://github.com/new
   → Создайте репозиторий (например: "telegram-miniapp")
   → НЕ отмечайте "Add a README file"
   → Нажмите "Create repository"
   → Скопируйте URL репозитория
   → Вставьте в скрипт

3. Проверка:
   → Запустите ПРОВЕРКА_НАСТРОЙКИ.bat снова
   → Должно быть: ✅ GitHub подключен

═══════════════════════════════════════════════════════════════

📌 ШАГ 2: ПОДКЛЮЧЕНИЕ NETLIFY (5 минут)

1. Откройте браузер
2. Зайдите на: https://app.netlify.com/
3. Войдите (или создайте аккаунт)
   → Можно войти через GitHub (рекомендуется)

4. Нажмите: "Add new site" → "Import an existing project"

5. Выберите: "GitHub"
   → Если первый раз - авторизуйтесь
   → Разрешите доступ к репозиториям

6. Выберите ваш репозиторий из списка

7. Настройки (ВАЖНО!):
   ┌─────────────────────────────────────┐
   │ Base directory:    webapp          │
   │ Build command:     (оставьте пустым)│
   │ Publish directory: .               │
   └─────────────────────────────────────┘
   
   Как настроить:
   → Нажмите "Show advanced"
   → Base directory: введите "webapp"
   → Build command: оставьте пустым
   → Publish directory: введите "." (точка)

8. Нажмите: "Deploy site"

9. Дождитесь завершения (1-2 минуты)
   → Вы увидите URL вашего сайта

10. Проверка:
    → Откройте URL сайта
    → Должен открыться ваш сайт
    → Нет ошибок? → Отлично!

📌 ШАГ 3: ЗАПУСК АВТОМАТИЧЕСКОГО ДЕПЛОЯ (1 минута)

ВАРИАНТ А: Запуск вручную
→ Запустите: ЗАПУСТИТЬ_АВТОДЕПЛОЙ_В_ФОНЕ.bat
→ Каждый раз при включении ПК

ВАРИАНТ Б: Автозапуск (РЕКОМЕНДУЕТСЯ)
1. Нажмите Win + R
2. Введите: shell:startup
3. Нажмите Enter
4. Скопируйте файл АВТОЗАПУСК_ПРИ_СТАРТЕ.bat
   в открывшуюся папку
   (или создайте ярлык)
5. Готово! Теперь запускается автоматически!

📌 ШАГ 4: ПРОВЕРКА РАБОТЫ (2 минуты)

1. Запустите: ПРОВЕРКА_НАСТРОЙКИ.bat
   → Должно быть: ✅ ВСЁ ОТЛИЧНО!

2. Тест автоматического деплоя:
   → Откройте файл: index.html
   → Найдите любую строку (например, заголовок)
   → Измените её (например, добавьте "TEST")
   → Сохраните файл (Ctrl+S)
   
   → Подождите 30 секунд
   → Откройте файл: auto_deploy.log
   → Должна быть запись: "📝 Изменения: index.html"
   → Должна быть запись: "✅ Деплой запущен"
   
   → Подождите ещё 1-2 минуты
   → Зайдите на GitHub
   → Должен быть новый коммит "Auto: ..."
   
   → Зайдите на Netlify
   → Должен быть новый деплой
   
   → Откройте Telegram мини-приложение
   → Обновите его (закройте и откройте)
   → Должны увидеть ваши изменения!

✅ Если всё работает → НАСТРОЙКА ЗАВЕРШЕНА!
