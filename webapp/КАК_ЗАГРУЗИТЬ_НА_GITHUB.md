# Как загрузить изменения на GitHub

## Вариант 1: Через GitHub Desktop (самый простой)

1. **Скачайте GitHub Desktop** (если еще не установлен):
   - Перейдите на https://desktop.github.com/
   - Скачайте и установите приложение

2. **Откройте GitHub Desktop**:
   - File → Add Local Repository
   - Выберите папку: `C:\Users\Alex\Desktop\Case`
   - Нажмите "Add repository"

3. **Загрузите изменения**:
   - Внизу слева введите описание изменений (например: "Добавлен загрузочный экран с анимацией GIFTS")
   - Нажмите кнопку "Commit to main"
   - Нажмите кнопку "Push origin" (вверху справа)

## Вариант 2: Через веб-интерфейс GitHub

1. **Откройте ваш репозиторий на GitHub.com**

2. **Загрузите файлы**:
   - Нажмите "Add file" → "Upload files"
   - Перетащите измененные файлы:
     - `webapp/index.html`
     - `webapp/style.css`
     - `webapp/app.js`
     - `webapp/images/splash_screen.jpg` (если добавили)
   - Внизу страницы введите описание: "Добавлен загрузочный экран"
   - Нажмите "Commit changes"

## Вариант 3: Установить Git и использовать командную строку

1. **Установите Git**:
   - Скачайте с https://git-scm.com/download/win
   - Установите с настройками по умолчанию

2. **Откройте командную строку (CMD)** в папке проекта:
   ```
   cd C:\Users\Alex\Desktop\Case
   ```

3. **Выполните команды**:
   ```bash
   git add .
   git commit -m "Добавлен загрузочный экран с анимацией GIFTS"
   git push
   ```

## После загрузки на GitHub

После того как изменения загружены на GitHub, Netlify автоматически обновит сайт (если настроена автоматическая синхронизация).

Если автоматическая синхронизация не настроена:
1. Зайдите на https://app.netlify.com/
2. Выберите ваш сайт
3. Нажмите "Trigger deploy" → "Deploy site"
